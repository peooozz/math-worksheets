# Kids Math Fun - Interactive Worksheet

A playful, interactive single-page web application designed for children aged 4-7 to practice early math skills.

## Features

### 1️⃣ Numbers Module (Tracing)
- **Interactive Tracing**: Trace numbers 0-10 with finger, mouse, or stylus.
- **Accuracy Detection**: Validates if the child stays on the path and covers the shape.
- **Stroke Order**: Visual guides (dots) and multi-stroke support for complex numbers.
- **Practice Grid**: Repeated numbers for visual reinforcement.
- **Fun Feedback**: Confetti and positive sounds upon completion.

### 2️⃣ Addition Module
- **Visual Learning**: Objects (apples, stars, bears, etc.) represent numbers.
- **Interactive Counting**: Tap objects to count them (1, 2, 3...).
- **Drag & Drop**: Playful interaction with objects.
- **Multiple Choice**: Select the correct answer from options.

### 3️⃣ Subtraction Module
- **Visual Take-Away**: Objects fade out to represent subtraction.
- **Equation Visualization**: See the math sentence clearly.
- **Interactive**: Engaging animations and sounds.

## Technical Details
- **Stack**: React, Vite, Tailwind CSS, Framer Motion.
- **Mobile-First**: Fully responsive, touch-optimized, prevents scrolling during interaction.
- **PWA-Ready**: Responsive viewport settings and manifest-ready structure.
- **No Backend**: Runs entirely in the browser.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Design Principles
- **Pastel Colors**: Gentle on the eyes.
- **Rounded UI**: Friendly and safe feel.
- **Positive Reinforcement**: Only positive feedback (confetti, stars), gentle hints for errors.
- **Accessibility**: Large touch targets, high contrast text.
